# RandomTreeCG
CG application randomly generating 3D tree with OpenGL
![circleanimationmuvie](demo.gif)

## Getting Started

You can run this system by this command on Mac OS.
```
./jogl.sh
```

### Prerequisites

Java

## Built With

* [OpenGL](https://www.opengl.org/) - Used the library to run CG systems.
